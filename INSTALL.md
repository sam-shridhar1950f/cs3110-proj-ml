## Setup & Installation

You'll need ANSITerminal to run this game, which can be installed with:
   ```
   opam install ansiterminal
   ```

### Playing the Game
To play the terminal-based game included in our project:
1. **Build the Program:**
   ```
   dune build
   ```
2. **Execute the Program:**
   ```
   dune exec bin/main.exe
   ```
